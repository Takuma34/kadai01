- ゴンとじゃんけんするゲームとなります。内容は単純なじゃんけんアプリとなりますが、
　画像の動きや画面の切り替えなど、ネットで調べながら、作成しました。

- 今回は、単純じゃんけんゲームを作成しましたが、次回は少し複雑なゲームを作成したいです。
